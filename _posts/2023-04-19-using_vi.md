## 변수 이름을 한꺼번에 바꾸고 싶을 때

esc를 누르고

**:%s/[바꾸고싶은문자열]/[바꿀문자열]/[옵션]**

```
:%s/babo/hehe/g
```

보통은 옵션에 g를 많이 쓴다.

g는 전체 파일에서 변경한다는 뜻

## 창을 두개 띄우고 싶을 때

esc를 누르고

**:new [파일경로]**

```
:new lib/babo.js
```

세로는

**:vs [파일경로]**

```
:vs lib/babo.js
```

### 창 간 커서 변경

**Ctrl + W**
